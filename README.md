# Negocio lomas de zamora

Este repositorio incluye analisis logisticos y una portada visual estilo liquid glass para el ruteo de repartos en Burzaco.

## Archivos de ruteo agregados

- `Rutas.xlsx`: base original de repartos e instituciones.
- `Rutas_ruteadas.xlsx`: archivo procesado con rutas por reparto.
- `index.html`: portada estilo liquid glass con acceso directo a los archivos.

## Ver rapido en local

```bash
python -m http.server 8080
```

Luego abrir `http://localhost:8080`.

## Informe interactivo existente

El informe completo con mapa se mantiene en `informe/public/index.html`.
